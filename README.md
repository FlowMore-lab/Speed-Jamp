# Doodle Jump 2 Clone 🎮

A simple clone of the popular Doodle Jump game built with Python and Pygame.

## 🎯 About the Game

Jump on platforms, climb higher, and try to beat your high score! The game gets progressively harder as you score more points.

## 🚀 Controls

- **← / A** - Move left
- **→ / D** - Move right
- **SPACE** - Restart after Game Over

## ⭐ Features

- ✅ 3 platform types:
  - 🟢 **Green** - Regular platforms
  - 🔵 **Blue** - Moving platforms (side to side)
  - 🔴 **Red** - Breaking platforms (collapse after 2 seconds, appear after 10000 points)
- ✅ 🚀 **Jetpack power-up** - Yellow item on platforms gives 2 seconds of flight
- ✅ 🎨 **Gradient background** - Beautiful color transition
- ✅ 📊 **Score & High Score** - Tracks your best run
- ✅ 📈 **Dynamic difficulty** - More moving and breaking platforms as you progress

## 📦 Installation

1. Make sure you have Python 3.7+ installed
2. Install Pygame.
